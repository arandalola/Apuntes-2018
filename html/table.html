Etiqueta table

Descripción:Tabla. Etiqueta de bloque. Sirve para mostrar datos.La etiqueta "caption" le da título a la tabla. Las etiquetas "tr" simbolizan las filas y "td" simbolizan las celdas de una tabla.

Para los encabezados de celdas y filas utilizamos la etiqueta "th", llevan de raíz un css en negrita.

Las etiquetas "thead" encabezado, "tbody" cuerpo y "tfoot" pie de tabla, sirven para diferenciar dentro de una tabla el encabezado, el contenido y el pie de tabla.

El atributo "colspan" combina celdas, el atributo se coloca en la línea y celda que corresponde y el atributo "rowspan" combina filas, el atributo se coloca en la línea y celda que corresponde, al combinar filas se restan celdas a las filas que comprende el atributo.
