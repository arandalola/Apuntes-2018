Etiqueta span

Descripción: Etiqueta de línea. Acota un texto, sin darle ningún tipo de notoriedad, para cambiar su formato utilizamos css. Para abreviar el simbolo "menor que" <span> < </span> se utiliza el código de escape (<) y para el simbolo "mayor que" <span> > </span> se utiliza el código de escape (>). Los códigos de escape sirven para que el navegador no interprete que su uso es una etiqueta sino su uso abreviado.
