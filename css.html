Regla CSS y vinculación

CSS significa Cascading Style Sheets. CSS es un lenguaje que describe el estilo de un documento HTML. CSS se utiliza para definir estilos para sus páginas web, incluyendo el diseño, la disposición y las variaciones en la pantalla para diferentes dispositivos y tamaños de pantalla.

El bloque de declaración contiene uno o más declaraciones separados por punto y coma. Cada declaración incluye un nombre de propiedad CSS y un valor, separados por dos puntos. Una declaración CSS siempre termina con un punto y coma, y ​​los bloques de declaraciones están rodeados por llaves.

La regla css que debes seguir para introducir una declaración es: selector, propiedad y valor. Los selectores CSS se utilizan para "encontrar" (o seleccionar) elementos HTML en función de su nombre del elemento, ID, clase, atributo, y mucho más.

	selector{
		propiedad:valor;
	}

Para ello tienes básicamente 3 opciones para aplicar la regla:

  1. Hojas de estilo CSS inline
  2. Hojas de estilo CSS internos
  3. Hojas de estilo CSS externos

La base de estos apuntes es la opción de hojas de estilo CSS externos. La etiqueta que se utiliza para enlazar un archivo html y uno .css es, <link rel="stylesheet" href="ruta hasta el archivo .css">, dentro de head.

Poco recomendable utilizar estilo inline para aplicar css, se utiliza style como atributo en las etiquetas, y entre comillas asignamos propiedad, dos puntos y valor, separados cada propiedad con su valor por punto y coma.

<head>

Propiedades básicas CSS

Los comentarios se utilizan para explicar el código, y puede ayudar al editar el código fuente en una fecha posterior. Los comentarios son ignorados por los navegadores.

Un comentario comienza con / * y termina con * /.

Se pueden introducir varias etiquetas en un selector, separadas por comas. Siempre que tengan propiedades en común las podemos agrupar de esta manera. Será mejor para agrupar los selectores, para minimizar el código.

Ejemplo:
p,h1,h5,{
	color:orange;    Los selectores indicados seran todos del mismo color.
}

El selector * selecciona todos los elementos del archivo html para darles propiedad y valor.

Ejemplo:
*{
	background-color:grey;  Esta propiedad da valor al fondo del sitio web,
				como utilizas el * como selector,
				referencia todos los elementos del documento.
}

Color

Los colores se especifican utilizando nombres de color predefinidos, o los valores RGB, HEX, HSL (tonalidad, saturación, luminosidad), RGBA (rojo, verde, azul, alfa) alfa especifica la opacidad de un color. HSLA (tonalidad, saturación, luminosidad, alfa).

Con CSS, un color generalmente se especifica mediante:

  • Un nombre de color válido - como "rojo"
  • Un valor HEX - como "# FF0000"
  • Un valor RGB (rojo, verde, y azul) - como "rgb (255,0,0)"

HTML es compatible con 140 nombres de colores estándar.

La propiedad color se utiliza para establecer el color del texto.

Ejemplo:
h1{
	color: green;
}

Tipografía y fuentes

La font-family de un texto se establece con la propiedad font-family.

La propiedad font-family debe contener varios nombres de fuentes como un sistema de "retroceso" en la declaración. Si el navegador no es compatible con la primera fuente, se intenta con el siguiente tipo de letra, y así sucesivamente. Si el nombre de una familia de fuentes es más de una palabra, debe ser entre comillas, como: "Times New Roman".

La forma de asignar una o varias familias tipográficas a un archivo .html, introduciendo una etiqueta link en el head, es la opción standard de Google Fonts..

Ejemplo:

<link href="https://fonts.googleapis.com/css?family=Butterfly+Kids" rel="stylesheet">

<link href="https://fonts.googleapis.com/css?family=Hanalei+Fill" rel="stylesheet">

.e2{
	font-family: 'Butterfly Kids', cursive;
}
.e3{
	font-family: 'Hanalei Fill', cursive;
}

El recurso Google Fonts con la opción @import, nos facilita asignar una o varias familias tipográficas en un archivo .css. Se asigna esta opción en el archivo .css al principio del documento sin ningún selector ni comentario, se recomienda probar las tildes y la ñ en la fuente seleccionada para comprobar que funciona correctamente para el idioma español. Con la opción Customize se pueden seleccionar todos los tipos de estilo que contenga la fuente.

Ejemplo:

@import url('https://fonts.googleapis.com/css?family=Baloo+Bhaina');

@import url('https://fonts.googleapis.com/css?family=Ubuntu+Mono:400,400i,700,700i');

h1{
	font-family: 'Baloo Bhaina', cursive;
	font-size:60px;
}
pre{
	font-family: 'Ubuntu Mono', monospace;
	font-size:20px;
}

La propiedad font-style se utiliza para especificar el estilo a la fuente del texto .

Ejemplo:
p.normal {
	font-style: normal;  Asigna un valor normal a la fuente del texto.
}
p.italic {
	font-style: italic;  Asigna un valor cursiva a la fuente del texto.
}
p.oblique {
	font-style: oblique;  Asigna un valor oblicuo a la fuente del texto.
}

La propiedad font-size establece el tamaño del texto. Se pueden utilizar otros valores de medida como em, en % y valores entre xx-large y xx-small.

El tamaño de la unidad em es recomendado por el W3C. 1 em es igual al tamaño de fuente actual. El tamaño de texto por defecto en los navegadores es 16px. Por lo tanto, el tamaño predeterminado de 1 em es 16px.

El procentaje % fija un tamaño de fuente en porcentaje para el elemento.

Se puede establecer el tamaño del texto mediante los valores, xx-large, x-large, large, medium, x-small y xx-small.

Ejemplo:
h1 {
	font-size: 40px;  
}
h2 {
	font-size: 2.5em;
}
h2 {
	font-size: 15%;
}
h2 {
	font-size: x-small;
}

Otras propiedades de fuente son font-variant, que con el valor small-caps convierte el texto seleccionado en mayúsculas; la propiedad font-weight, establece cómo se deben mostrar caracteres gruesas o delgadas en el texto, los valores que podemos utilizar son un rango del 100 al 900, de 100 en 100, o bold, bolder, lighter, initial, inherit. Por defecto el valor será normal.

La propiedad font, engloba varios valores y así se reduce el código. A continuación se muestra un ejemplo de cómo mostrar la declaración mediante "short hand", el orden que deben seguir y los valores que son obligatorios, que son font-family y font-size, los demás son todos opcionales.

Ejemplo:
.font{
	font:font-style font-variant font-weight font-size/line-height font-family;
}
				.font{
	font:italic small-caps bold 30px/1.5 Ewert;
}

Texto

La propiedad text-align se utiliza para establecer la alineación horizontal de un texto, siempre que el elemento sea de línea, también se utiliza cuando una imagen está dentro de un contenedor. Cuando el elemento a centar sea de blque se utiliza margin:0 auto; Un texto puede ser alineado a la izquierda o la derecha, centrado o justificado.

Ejemplo:
h1{
	text-align: center;  Texto alineado en el centro.
}
h2{
	text-align: left;  Texto alineado a la izquierda.
}
h5{
	text-align: right;  Texto alineado a la deracha.
}
h4{
	text-align: justify;  Texto justificado. No se utiliza.
}

La propiedad text-decoration se utiliza para establecer o eliminar decoraciones de texto.

Ejemplo:
a{
	text-decoration: none;  	Este valor se utiliza a menudo para eliminar los subrayados de enlaces.
}
h1 {
	text-decoration: overline;  	Valor que dibuja una línea encima del texto.
}
h2 {
	text-decoration: line-through;  Valor que tacha el texto.
}
h5 {
	text-decoration: underline;  	Valor que dibuja una línea bajo el texto.
}

La propiedad text-transform se utiliza para especificar letras mayúsculas y minúsculas en un texto.

Ejemplo:
p.uppercase {
	text-transform: uppercase;  Transforma el texto en mayúsculas.
}
p.lowercase {
	text-transform: lowercase;  Transforma el texto en minúsculas.
}
p.capitalize {
	text-transform: capitalize;  Transforma las primera letras de una palabra en mayúscula.
}

La propiedad text-shadow le da sombra al texto.

Ejemplo:
h1 {
	text-shadow: 3px 2px red; Asigna un valor que le da sombra al texto. 1º valor desplaza hacia la derecha,
				  2º valor desplaza hacia abajo y 3º valor le asigna color al difuminado.
}

La propiedad text-indent le da tabulado al texto. El valor se puede asignar en px, em o %.

Ejemplo:
h1 {
	text-indent:5px; Asigna tabulado al texto.
}

Otras propiedades de texto son letter-spacing, que da espacio específico entre letras, su valor se asigna normalmente con píxels; line-height, el interlineado de los párrafos, su valor se asigna con varios tipos de medida, como em, % o píxels.

Altura y anchura

Las propiedades height y width se utilizan para ajustar la altura y la anchura de un elemento. El height y width se pueden configurar para especificar en valores de longitud , como px, cm, etc., o en porcentaje (%).

Las propiedades height y width no incluyen el padding, margin o bordes.

Al configurar las propiedades de anchura y altura de un elemento con CSS, teniendo en cuenta el ancho y la altura del área de contenido, se debe calcular el tamaño de un elemento, también debe agregar márgenes interiores, bordes y márgenes exteriores.

Suponemos que queremos un elemento <div> que tenga una anchura total de 350 píxeles:

Ejemplo:
div {
	width: 320px;		Propiedad que da valor al ancho del elemento en px.
	height: 160px;		Propiedad que da valor a la altura del elemento en px.
	padding: 10px;  	Propiedad que referencia los valores interiores del margen.
	border: 5px solid grey; 1º valor el grosor, 2º valor el estilo y 3º valor el color.
	margin: 0;  		Propiedad que referencia los valores exteriores del margen.
}
350px x 190px

Aquí está el cálculo:

  • 320px (anchura)
  • + 20 píxeles (izquierda + derecha relleno)
  • + 10px (izquierda + derecha frontera)
  • + 0px (margen izquierdo + derecho)
  • = 350 píxeles

La anchura total de un elemento debe calcularse así:

Ancho total elemento = ancho + acolchado izquierda + relleno derecho + borde izquierdo + borde derecho + margen izquierdo + margen derecho.

La altura total de un elemento debe calcularse así:

Altura total elemento = altura + top acolchado + acolchado inferior + borde superior + borde inferior + margen superior + margen inferior.

Propiedades CSS

ID

El selector de ID utiliza el atributo id de un elemento HTML para seleccionar un elemento específico. La identificación de un elemento debe ser único dentro de una página, por lo que el selector de ID se utiliza para seleccionar un elemento único! Para seleccionar un elemento con un ID específico, escribe un carácter almohadilla (#), seguido por el id del elemento.

Nota: Un nombre de ID no puede comenzar con un número!

Ejemplo:
#miid{
	text-transform:uppercase;  Propiedad que transforma a mayúsculas el texto.
}

Class

El selector de clase selecciona elementos con un atributo de clase específica. Para seleccionar elementos con una clase específica, escriba un punto (.) seguido del nombre de la clase. Funciona como un id, pero el nombre de la clase se puede repetir tantas veces se necesite.

Nota: Un nombre de clase no puede comenzar con un número!

Ejemplo:
.parrafospares{
	color:blue;  Propiedad que da color al texto.
}

Descendientes

Para dar estilo a etiquetas que se repiten en el documento html y diferenciarlas unas de otras, utilizamos la descendencia, se indica la etiqueta que queremos diferenciar, separando cada nivel de descendencia por un espacio. Indicando la etiqueta de la que desciende y la etiqueta descendiente.

Si tenemos una propiedad y valor común para diferentes selectores, podemos agrupar del siguiente modo: header span, main span{propiedad y valor en común} o header span, main em, footer strong{propiedad y valor en común}.

Ejemplo:
header p{
	color:orange;  Propiedad que da color al texto.
}
main p{
	color:blue;    Propiedad que da color al texto.
}
footer p{
	color:green;   Propiedad que da color al texto.
}

Descendientes directos

Para referenciar etiquetas que son descendientes directos, utilizamos en el selector: la etiqueta padre seguida del símbolo >, seguido de un espacio y la etiqueta descendiente directa.

Ejemplo:
header> p{
	color:#0f0;  Propiedad que da color al texto con el método hexadecimal en el valor.
}
main div> p{
	color:red;   Propiedad que da color al texto.
}

Primer hermano

El primer hermano hace referencia al hermano directo de la etiqueta en el documento html. En primer lugar en el selector el hermano mayor, seguido del símbolo +, y de la etiqueta que es primer hermano, seguido después de un espacio por la clase o id al que queremos referenciar, si los hubieran.

Ejemplo:
h2+p .inicial, p+p .inicial{ 
	font-size:48px;  Propiedad que da tamaño al texto en px.
	color:#FFF;      Propiedad que da color al texto con el método hexadecimal en el valor.
}

Todos los hermanos

Todos los hermanos referencia a todos los hermanos de una etiqueta en el documento html. En primer lugar en el selector el hermano mayor, seguido del símbolo ~, y de la etiquetas que son hermanos, seguido después de un espacio por la clase o id al que queremos referenciar, si los hubieran.

Ejemplo:
h2~p .inicial, p~p .inicial{ 
	font-size:48px;  Propiedad que da tamaño al texto en px.
	color:#FFF;      Propiedad que da color al texto con el método hexadecimal en el valor.
}

Margin y Padding

El margin hace referencia al margen exterior de la caja y el padding referencia al margen interior de la caja. Denominamos los lados de la caja con top(arriba), right(izquierda), bottom(abajo)y left(derecha) y las colocamos en el valor en sentido de las manillas del reloj. Si ponemos un solo valor (00px) referencia a todos con el mismo valor, si ponemos dos valores el primero referencia a top y bottom, y el segundo a right y a left, si ponemos tres valores el primero referencia a top, el segundo a right y left y el tercero referencia al bottom.

Si elegimos una imagen con fondo transparente se visualiza mejor el efecto del margin y padding. Los valores negativos son permitidos sólo en el margin.

Para mantener la anchura de la caja cuando utilizamos padding,ya que el padding añadido a dicho elemento será añadido a la anchura total de la caja, puede utilizar la propiedad box-sizing:border-box;. Esto hace que el elemento mantenga su anchura; si aumenta el relleno, el espacio contenido disponible disminuirá.

Esquema margin y padding

Explicación de las diferentes partes:

  • Contenido. El contenido de la caja, donde aparecen textos e imágenes.
  • Padding. Borra un área alrededor del contenido. El padding es transparente.
  • Border. Un borde que va alrededor del padding y contenido.
  • Margin. Borra un área fuera del borde. El margin es transparente.
Ejemplo:
p{
	margin:30px 10px 40px 15px;  Propiedad que referencia los valores exteriores del margen, top-right-bottom-left
	padding:20px 20px 50px 20px; Propiedad que referencia los valores interiores del margen, top-right-bottom-left
}
				
main img{
	margin:10px;  		Si damos un valor, referencia a todos los margenes exteriores.
	margin:10px 10px;  	Si damos dos valores, el primer valor referencia al margen top y bottom exteriores, 
				y el segundo valor a right y left exteriores.
	margin:10px 10px 10px;  Si damos tres valorer, el primer valor referencia al margen top exterior, 
				el segundo a right y left exteriores, y el tercero a bottom exterior.
	padding:20px;  		Si damos un valor, referencia a todos los margenes interiores.
	padding:10px 10px;  	Si damos dos valores, el primer valor referencia al margen top y bottom interiores, 
				y el segundo valor a right y left interiores.
	padding:10px 10px 10px; Si damos tres valorer, el primer valor referencia al margen top interior, 
				el segundo a right y left interiores, y el tercero a bottom interior.
}

Con la propiedad y valor, margin:auto; se centran horizontalmente los elementos en su caja.

main div{
	width:300px;   Propiedad que da valor al ancho de la caja en px.
	padding:40px;  Propiedad que da valor a todos los margenes interiores en px. 
	margin:auto;   Propiedad que centra los elementos de bloque.
}

Border

Los CSS border, propiedades que permiten especificar el estilo, el ancho y el color del borde de un elemento. En la propiedad se puede referenciar especificamente el width, el style y el color, colocando el valor en sentido de las manillas del reloj. También se puede referenciar especificamente que borde de la tabla quieres modificar, top, right, bottom o left.

El valor de style, se debe indicar porque en su defecto selecciona none, y entonces no aparece ningún borde. Si indicamos solo el style, el width por defecto vale 1px y el color por defecto es el negro.

Ejemplo:
#multiple{
	border-color:red green black blue; Valor de color top-right-bottom-left
	border-style:solid dotted dashed;  Valor de estilo top-(right-left)-bottom
	border-style:solid dotted;         Valor de estilo (top-bottom)-(right-left)
	border-width:20px;                 Valor de ancho que referencia a todos los lados.
}
				
#multiple{
	border-top:5px solid #f00;  Referencia el borde de arriba,1º valor el grosor, 2º valor el estilo y 3º valor el color.
	border-bottom:5px solid;    Referencia el borde de abajo, 1º valor el grosor y 2º valor el estilo del borde, 
				    el color por defecto es el negro.
	border-left:solid #f00;     Referencia el borde izquierdo, 1º valor el estilo y 2º valor el color del borde, 
				    el grosor por defecto es de 1 px.
	border-right:solid;         Referencia el borde derecho, valor de estilo de borde, por defecto el grosor es 1px
				    y el color es negro.
}

	border-radius:15px 0px 15px 0px;  La propiedad border-radius redondeamos las esquinas de la caja,
	                                  dando los valores en el sentido de las manillas del reloj, top(00px),

}

}

}
				
#dotted{
	border:5px dotted #f00;  Con el style dotted el borde aparece con una línea de puntos.
}
				
#double{
	border:5px double #f00;  Con el style double el borde aparece con doble línea.
}
				
#dashed{
	border:5px dashed #f00;  Con el style dashed el borde aparece con una línea discontinua.
}
				
#groove{
	border:5px groove #f00;  Define un borde acanalado 3D.

#ridge{

Outline

La propiedad outline, hace que el perfil no sume la unidad de medida utilizada, en este caso pixels, al elemento.

Ejemplo:
.perfil{
	outline:1px solid #000;  1º Referencia el grosor de los bordes, 2º valor el estilo y 3º valor el color.
	outline:none; 		 Hace que no aparezca el borde.
}

También podemos aplicar outline a un texto para definirlo o destacarlo dentro del cuerpo y para que no se visualice el borde.

Box-shadow

Sombreado para cajas, es una caja superpuesta que con los dos primeros valores la desplazamos, con el tercer valor le damos densidad al difuminado y el cuarto valor le da color al difuminado.

Ejemplo:
#difright{
	width:400px;   			 Propiedad que referencia el ancho de la caja en px.
	height:250px;  			 Propiedad que referencia el alto de la caja en px.
	border:solid;  		  	 Estilo del borde de la caja.
	margin:100px auto 100px;  	 Bordes exteriores de la caja.
	box-shadow:20px 20px 30px #000;  Sombreado right inferior de la caja.				
	1ºpx desplaza a la derecha - 2ºpx desplaza abajo - 3ºpx densidad difuminado - 4ºpx color del difuminado
}
				
#difleft{
	width:400px;

Background

Las propiedades de fondo CSS, background, se utilizan para definir los efectos de fondo de los elementos.

A continuación vemos varios tipos de background:

Backgroung linear-gradient, crea una transición suave y lineal de al menos dos colores. También puede establecer un punto de partida y una dirección (o un ángulo).

Ejemplo:
.vertical{
	width:200px;
	height:200px;
	border:solid;
	margin-bottom:30px;
	background:linear-gradient(blue,purple,green,yellow,red);	      Gradient lineal - de arriba a abajo 
	(este valor es por defecto).
}
.horizontal{
	width:200px;
	height:200px;
	border:solid;
	margin-bottom:30px;
	background:linear-gradient(to right,blue,purple,green,yellow,red);    Gradient Lineal - De izquierda a derecha.
}
.diagonal{
	width:200px;
	height:200px;
	border:solid;
	background:linear-gradient(to top left,blue,purple,green,yellow,red); Gradient lineal que comienza en la parte 
									      superior izquierda y va a la parte 
									      inferior derecha.
}

Background-color, propiedad que especifica el color de fondo de un elemento. Con el valor RGBA (rojo, verde, azul, alfa) alfa especifica la opacidad de un color.

Ejemplo:
.exterior{
	width:500px;
	height:500px;
	border:solid;
	background-color:rgb(0,0,255);  Valor definido en rgb (rojo, verde, y azul).
}
.interior{
	width:250px;
	height:250px;
	border:solid;
	margin:125px;
	background-color:rgb(0,255,0);
}
.exterior1{
	width:500px;
	height:500px;
	border:solid;
	background-color:rgb(0,0,255);
	margin-top:20px;
	margin-bottom:20px;
}
.interior1{
	width:250px;
	height:250px;
	border:solid;
	margin:125px;
	background-color:rgba(0,255,0,0.6);  Valor definido en rgb (rojo, verde, azul, alfa).
}

Background-image, propiedad que especifica una imagen para usar como el fondo de un elemento.

Ejemplo:
.image{
	background-repeat:no-repeat;  			Valor que hace no repetir la imagen.
	background-size:cover;	      			Valor que hace cubrir toda la caja con la imagen.
	background-image:url("../imagenes/image.jpg");  Propiedad que referencia una imagen con valor url.
}
.image:hover{
	background-repeat:no-repeat;
	background-size:100% 100%;  Valor en % que hace cubrir toda la caja con la imagen.
	background-image:url("../imagenes/image1.jpg");
}

Backgroung-repeat. Por defecto, la imagen se repite de manera que cubra todo el elemento, tanto horizontal como verticalmente. Otros valores que le podemos dar son: "no-repeat", que no repite la imagen. "repeat-x" que repite en el eje horizontal y "repeat-y" que repite en el eje vertical.

Ejemplo:
.repeat{
	background-image:url("../imagenes/image100.jpg");
	background-repeat:repeat;  Valor por defecto que repite la imagen.
}
.no-repeat{
	background-image:url("../imagenes/image1100.jpg");
	background-repeat:no-repeat;  Valor que hace no repetir la imagen.
}
.x{
	background-image:url("../imagenes/image100.jpg");
	background-repeat:repeat-x;  Valor que repite la imagen en el eje horizontal.
}
.y{
	background-image:url("../imagenes/image1100.jpg");
	background-repeat:repeat-y;Valor que repite la imagen en el eje vertical.
}

Backgroung-position, especifica la posición de la imagen. Consta de dos valores, el primer valor referencia a right y left, y el segundo valor a top y bottom. "center center" nos centra la imagen, también podemos untilizar % y px. Se pueden combinar varios valores para mostrar la imagen en cualquier lado de la caja.

Ejemplo:
.position{
	background-image:url("../imagenes/image1100.jpg");
	background-position:center right;  El primer valor referencia a right y left, y el segundo valor a top y bottom.
	background-repeat:no-repeat;
}

Backgroung-attachment, especifica que la imagen de fondo sea fija o no (no se desplazará con el resto de la página)

Ejemplo:
.attachment{
	background-image:url("../imagenes/image1100.jpg");
	background-position:left top;
	background-repeat:no-repeat;
	background-attachment:scroll;  Valor por defecto, la imagen no se mueve con el movimiento del scroll.
}
.attachment{
	background-image:url("../imagenes/image1100.jpg");
	background-position:right top;
	background-repeat:no-repeat;
	background-attachment:fixed;  Valor que hace que la imagen quede fija con el movimiento del scroll.
}

Background-size, ajusta la imagen imagen de fondo a la caja.

Ejemplo:
.cover{
	background-repeat:no-repeat; 
	background-size:cover;  Valor que ajusta las imágenes que son más pequeña que la caja donde se encuentran.
	background-image:url("../imagenes/image.jpg");
}
.contain{
	background-repeat:no-repeat; 
	background-size:contain;  Valor que ajusta las imágenes que son más grandes que la caja donde se encuentran.
	background-image:url("../imagenes/image.jpg");
}

Background-origin, específica la posición de origen de una imagen de fondo.

Ejemplo:
.padding{
	background-image:url("../imagenes/image1.jpg");
	background-size:cover;
	background-repeat:no-repeat; 
	background-origin:padding-box;  Valor que posiciona la imagen desde el borde de la caja interior hacia dentro.
}
.border{
	background-image:url("../imagenes/image1.jpg");
	background-size:cover;
	background-repeat:no-repeat; 
	background-size:cover;
	background-origin:border-box;  Valor que posiciona la imagen desde el borde exterior de la caja hacia dentro. 
}
.content{
	background-image:url("../imagenes/image1.jpg");
	background-size:cover;
	background-repeat:no-repeat; 
	background-size:cover;
	background-origin:content-box;  Valor que posiciona la imagen desde el padding que le asignamos. 
}

Background-clip, específica hasta qué punto, el fondo debe extenderse dentro de una caja.

Ejemplo:
.image1{
	background-color:rgb(0,0,255);
	background-clip:padding-box;  Valor que posiciona el color de la caja desde el borde de la caja interior hacia dentro.
}
.image2{
	background-color:rgb(0,0,255);
	background-clip:border-box;  Valor que posiciona el color de la caja desde el borde exterior de la caja hacia dentro. 
}
.image3{
	background-color:rgb(0,0,255);
	background-clip:content-box;  Valor que posiciona el color de la caja desde el padding que le asignamos. 
}

Se pueden referenciar dos o más rutas de imagen. Se colocan las rutas separadas por una coma.

Ejemplo:
.variasImagenes{
	background-image:url("../imagenes/tierra.png"),
	url("../imagenes/agua.png"),url("../imagenes/viento.png"),
	url("../imagenes/fuego.png"),	url("../imagenes/ball.png");
	background-repeat:no-repeat;
	background-position:left top, right top, right bottom, left bottom, center;
}

Display

La propiedad display especifica el comportamiento de presentación de un elemento. El valor de la propiedad de visualización por defecto es en línea. Con los diferentes valores que se pueden asignar a la propiedad; block, inline, none, etc. hace que el comportamiento de los elementos cambie, por ejemplo un elemento de línea pase a bloque o al revés.

Visibility

Propiedad que especifica si un elemento es visible o no. El valor por defecto es visible y con hidden hace que el elemeto sea invisible, pero el resto de elementos respetan su espacio.

Overflow

La propiedad overflow especifica qué debe ocurrir si el contenido sobresale de la caja de un elemento. Esta propiedad especifica si se recorta el contenido o añadir barras de desplazamiento cuando el contenido de un elemento es demasiado grande para caber en un área especifica. Sólo funciona para elementos de bloque con una altura especifica.

Los diferentes valores que se pueden utilizar son: visible, hidden, scroll, auto, etc.

Posicionamiento

Position

Propiedad que especifica el tipo de método de posicionamiento utilizado para un elemento. Los valores que se pueden asignar son: static, por defecto; relative, absolute, fixed, sticky, etc.

Opacity

La propiedad opacity establece el nivel de opacidad de un elemento. El nivel de opacidad describe el nivel de transparencia, donde 1 no es transparente en absoluto, 0,5 es 50% ver a través, y 0 es completamente transparente.

Z-index

Propiedad que especifica el orden de lugar de un elemento. Un elemento con mayor orden de lugar está siempre en frente de un elemento con un orden de lugar inferior. Z-index sólo funciona en elementos posicionados, posición absoluta, relativa, o fija.

Float

La propiedad float especifica cómo un elemento debe flotar. Los elementos pasan a otra dimensión, hacia la izquierda (left), hacia la derecha (right) o por defecto sin flotar (none).

Animaciones

La animación CSS permite la animación de la mayoría de los elementos HTML sin necesidad de utilizar JavaScript.

Una animación permite a un elemento cambiar gradualmente de un estilo a otro. Se puede cambiar tantas propiedades CSS que se desee. Para utilizar la animación CSS, primero debe especificar algunos fotogramas clave para la animación, como especificar estilos CSS dentro de la norma @keyframes, la animación cambiará gradualmente desde el estilo actual al nuevo estilo en determinados momentos.

Se pueden asignar varias propiedades para ejecutar la animación, por ejemplo: animation-delay, animation-direction, animation-duration, animation-timing-function, etc.

Responsive

Diseño web adaptable o Responsive Web Design (RWD). El objetivo del diseño web adaptable es adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visitarlas. Hoy día las páginas web se ven en multitud de dispositivos como tabletas, teléfonos inteligentes, libros electrónicos, portátiles, PC, etcétera. Además, aún dentro de cada tipo, cada dispositivo tiene sus características concretas: tamaño de pantalla, resolución, potencia de CPU, sistema operativo o capacidad de memoria entre otras. Esta tecnología pretende que con un único diseño web, todo se vea correctamente en cualquier dispositivo.
